# Requirements and traceability
- R1 (MUST): The system MUST enforce lexicographic guardrails in the order Tier 1 through Tier 5. (Acceptance: AC-1)
- R2 (MUST): Each run MUST generate a Decision Capsule that enables deterministic replay. (Acceptance: AC-2)
- R3 (MUST): The system MUST emit a feasibility certificate or infeasibility diagnostic for all decisions. (Acceptance: AC-3)
- R4 (SHOULD): The system SHOULD support WDRO-CVaR primary mode and scenario-robust fallback. (Acceptance: AC-4)
- R5 (MAY): The system MAY provide governance override workflows via override capsules. (Acceptance: AC-5)
